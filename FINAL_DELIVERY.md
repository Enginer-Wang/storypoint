## 🚀 部署说明

### 最小化部署步骤

1. **安装依赖**
   ```bash
   npm install
   ```

2. **编译代码**
   ```bash
   npm run build
   ```

3. **启动服务**
   ```bash
   npm start
   ```

4. **访问应用**
   - PC 端: `http://localhost:4000`
   - 手机端: 扫码或访问 `http://localhost:4000/join/[SESSION_ID]`

### 验证部署成功

启动成功时会看到:
```
Connected to SQLite database
Database initialized
Server running at http://localhost:4000
PC端访问: http://localhost:4000/
```

## 💡 使用流程示例

### 场景: Sprint Grooming 会议

```
1. PM 在 PC 打开应用
   ↓
2. 输入故事号 "STORY-001-用户登录"
   ↓
3. 点击"开始估算"生成二维码
   ↓
4. 团队成员用手机扫码加入
   ↓
5. 每个成员选择自己的估算点数
   ↓
6. 成员点击"提交估算"
   ↓
7. PM 看到所有人已提交，点击"完成估算"
   ↓
8. 系统显示:
   - FE 平均值: 5 点
   - BE 平均值: 8 点
   - QA 平均值: 5 点
   - 建议最终: 6 点
   ↓
9. PM 编辑最终决策
   - 最终点数: 8 点
   - 预计工时: 1.5 小时
   ↓
10. 点击"保存决策"完成
```

## 📞 获取帮助

### 常见问题解决

**Q: 无法访问 localhost:4000**
A: 确保 npm start 已成功启动，检查防火墙设置

**Q: 二维码无法扫描**
A: 增加浏览器窗口大小或使用手机更接近

**Q: 手机无法加入**
A: 确保 PC 和手机在同一网络

**Q: 数据丢失**
A: 删除 data/story_point.db 后重启应用
